# Offer Builder

Проект переведен в формат HR Offer Builder с двумя экранами:

- `offer-builder/index.html` — разводящая страница (избранные + все офферы)
- `offer-builder/editor.html` — редактор оффера с автосохранением, дублированием, удалением и экспортом PDF

## Локальный запуск

```bash
npm run site:serve
```

Открыть:

- `http://localhost:4173/offer-builder/index.html`

## Что реализовано

- Автосохранение всех изменений в `localStorage`
- Независимый скролл сайдбара и превью на desktop
- Адаптивный режим с вертикальным стеком на узком viewport
- Дублирование шаблона с префиксом `(Копия) `
- Избранное и сортировка по последнему сохранению
- Динамические условия ЗП (добавление/удаление)
- Для страны `Армения` скрывается блок НДФЛ в превью
- Удаление оффера через модальное подтверждение
- Экспорт текущего оффера в PDF
