# Badge spec

- Source Figma node: `3480:5323`
- Component supports:
- `style`: `fill` | `outline`
- `size`: `L` | `M` | `S` | `XS`
- `device`: `desktop` | `mobile`
- Optional parts:
- `leftIcon`
- `rightIcon`
- `text2` (secondary text block with divider)
- Colors must be token-driven:
- text color
- fill color (for `fill`)
- stroke color (for `outline`)
- divider color
- Default icon: `chip`
- Uses typography tokens:
- `L – Med`, `S – Med`, `XS – Med`, `XXS – Med`
